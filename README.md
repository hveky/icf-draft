## 它解决什么问题

在 SSU 等康复工作场景里，**ICF 功能评估的修订与报告撰写**是高频、重复、又格式严苛的工作：
要对照 ICF 限定值与类目逐条核对、按规范措辞反复修订、保证术语统一——既费时间，又容易在繁琐中出错。

**icf-draft 的目标，是把这部分撰写工作自动化**：让评估者把时间花在判断与患者身上，而不是文字的搬运与格式的打磨。

## 🎯 目标用户

- 康复科医生、治疗师与评估人员
- 需要按 ICF 框架产出/修订功能评估报告的 SSU 工作者
- 关注康复信息化、想减少文书负担的临床团队

## ✨ 设想中的能力

> 以下为产品方向，随开发推进逐步落地。

- 📝 **结构化修订** —— 按 ICF 类目与限定值规范，辅助生成与修订评估文字。
- 🔁 **减少重复劳动** —— 把高频、模板化的撰写交给自动化，人只做确认与判断。
- 📑 **术语与格式一致** —— 统一措辞与排版，降低人工核对成本。
